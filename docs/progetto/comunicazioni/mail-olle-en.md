**Subject:** Validation of the QSA and CounselorBot project at KTH

Dear Olle,

I would like to propose an activity to do at KTH. It involves validating a self-assessment questionnaire called QSA and using a chatbot I built to analyze the results together with the students.

**The questionnaire is called QSA — Learning Strategies Questionnaire** (*Questionario sulle Strategie di Apprendimento*). It is a **self-assessment tool** with which students evaluate their own study strategies. It provides a profile across 14 dimensions, including **cognitive strategies** (e.g., self-regulation, concentration) and **affective-motivational factors** (e.g., anxiety, motivation). It was developed by **Michele Pellerey** and is widely used in Italy, including at the university level.

The work consists of two parts:

**Validation in English.** I already have a tool ready to **administer the questionnaire online** and to **validate it** (factor analysis, reliability, normative thresholds). If desired, we could also validate it in Swedish, but I don't know if it's worth it.

**CounselorBot.** This is a project of mine: after the test, the student receives their profile and can dialogue with an AI tutor that, using a Socratic approach, helps them reflect on their weaker areas and improve their study method.
The link to the project is here: https://counselorbot-sbs.ai4educ.org/. I have created access credentials for you: balter@kth.se pw. Olle2026.

I wanted to ask you: is this something you might be interested in? And do you think it would be feasible to involve KTH, perhaps through a seminar or an activity with the students?

If it's feasible, we can talk about it whenever you like.
